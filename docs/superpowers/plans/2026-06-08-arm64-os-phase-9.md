# MyOSv2 — Phase 9 Implementation Plan (fork + copy-on-write)

> REQUIRED SUB-SKILL: superpowers:executing-plans. Checkboxes track steps.

**Goal:** `fork()` with copy-on-write address-space duplication.

**Architecture:** `as_clone` shares writable pages read-only + COW and bumps refcounts; a write fault calls `cow_fault` to copy the page; `SYS_FORK` builds a child thread that resumes at the parent's `svc` with x0=0 via a `fork_ret` trampoline.

**Tech Stack:** Freestanding C + AArch64, the Phase 6b VM, the Phase 8 process model.

**Note (test-first):** COW page mechanism is unit-tested before wiring fork. `make test` green; gate active.

## Task 1: Page refcounts + `as_clone` + `cow_fault` (TEST FIRST)

**Files:** `src/vm.h`/`vm.c`, `src/tests.c`

- [ ] **Step 1: Add 5 COW tests to `src/tests.c`** (before `tests[]`):

```c
static void test_cow_clone_shares_pages(void)
{
    pmm_init(); kheap_init(); vm_init();
    struct addrspace *p = as_create();
    struct addrspace *c = as_clone(p);
    KASSERT(as_translate(p, USER_DATA_VA) == as_translate(c, USER_DATA_VA));
    KASSERT(as_translate(c, USER_DATA_VA) != 0);
}
static void test_cow_clone_refcount(void)
{
    pmm_init(); kheap_init(); vm_init();
    struct addrspace *p = as_create();
    struct addrspace *c = as_clone(p);
    uint64_t pa = as_translate(p, USER_DATA_VA);
    KASSERT(page_refcount(pa) == 2);
}
static void test_cow_fault_copies(void)
{
    pmm_init(); kheap_init(); vm_init();
    struct addrspace *p = as_create();
    // write a marker into the parent's data page (parent's mapping, still RW here)
    uint64_t ppa = as_translate(p, USER_DATA_VA);
    *(volatile uint8_t *)(uintptr_t)ppa = 0x5A;
    struct addrspace *c = as_clone(p);
    KASSERT(cow_fault(c, USER_DATA_VA) == 1);
    uint64_t cpa = as_translate(c, USER_DATA_VA);
    KASSERT(cpa != ppa);                                  // now private
    KASSERT(*(volatile uint8_t *)(uintptr_t)cpa == 0x5A); // content preserved
}
static void test_cow_fault_refcount(void)
{
    pmm_init(); kheap_init(); vm_init();
    struct addrspace *p = as_create();
    uint64_t pa = as_translate(p, USER_DATA_VA);
    struct addrspace *c = as_clone(p);
    KASSERT(page_refcount(pa) == 2);
    cow_fault(c, USER_DATA_VA);
    KASSERT(page_refcount(pa) == 1);                      // child took its own copy
}
static void test_cow_fault_non_cow(void)
{
    pmm_init(); kheap_init(); vm_init();
    struct addrspace *p = as_create();
    KASSERT(cow_fault(p, 0x999000UL) == 0);              // unmapped/non-COW
}
```

Register:
```c
    { "cow: clone shares pages",          test_cow_clone_shares_pages },
    { "cow: clone refcount 2",            test_cow_clone_refcount },
    { "cow: fault copies page",           test_cow_fault_copies },
    { "cow: fault drops refcount",        test_cow_fault_refcount },
    { "cow: fault on non-cow -> 0",       test_cow_fault_non_cow },
```
