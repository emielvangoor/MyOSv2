### Task 2: virtio-mmio transport + virtqueue (`virtio.c`)

**Files:** `src/virtio.h`, `src/virtio.c`

- [ ] **Step 1 — `virtio.h`:** register offsets, status/feature/desc flags,
`struct virtq { volatile struct virtq_desc *desc; volatile uint16_t *avail;
volatile uint16_t *used; uint32_t num; uint16_t last_used; };`, and
`uint64_t virtio_find(uint32_t device_id);` (returns a slot base or 0),
`int virtio_setup_queue(uint64_t base, struct virtq *q);`,
`int virtq_submit(uint64_t base, struct virtq *q, struct vbuf {uint64_t addr; uint32_t len; int write;} *bufs, int n);`.

- [ ] **Step 2 — `virtio.c`:**
- `virtio_find(id)`: scan `0x0a000000 + i*0x200` for i in 0..31; check Magic
  `0x74726976`, Version 2, DeviceID == id; return the base or 0.
- `virtio_setup_queue`: reset; `ACKNOWLEDGE|DRIVER`; feature negotiation accepting
  bit 32 (VERSION_1); `FEATURES_OK` (verify); QueueSel=0; read QueueNumMax (cap to
  8); QueueNum=8; allocate desc/avail/used (3 pmm pages, zeroed); write the three
  ring addresses (low/high); QueueReady=1; `DRIVER_OK`. Fill `*q`.
- `virtq_submit`: write `n` descriptors (chained; `WRITE` flag per `bufs[i].write`,
  `NEXT` on all but the last); `avail->ring[avail->idx % num] = 0`; `dsb`;
  `avail->idx++`; `dsb`; write QueueNotify=0; poll `used->idx != q->last_used`;
  `q->last_used = used->idx`. Return 0.

(MMIO via `volatile uint32_t *`; 64-bit ring addresses split into low/high 32.)

- [ ] **Step 3:** builds (linked but unused yet). Commit + push
`feat(virtio): virtio-mmio transport + polled split virtqueue (Phase 19)`.

### Task 3: virtio-blk + block API + tests

**Files:** `src/virtio_blk.c`, `src/block.h`, `src/kmain.c`, `src/tests.c`

- [ ] **Step 1 — failing tests** (run under `make test`, disk attached):
```c
static void test_block_present(void)
{
    pmm_init();
    virtio_blk_init();
    KASSERT(block_present());
}
static void test_block_write_read(void)
{
    pmm_init();
    virtio_blk_init();
    static uint8_t w[512], r[512];
    for (int i = 0; i < 512; i++) { w[i] = (uint8_t)(i * 7 + 3); r[i] = 0; }
    KASSERT(block_write(1, w) == 0);
    KASSERT(block_read(1, r) == 0);
    for (int i = 0; i < 512; i++) { KASSERT(r[i] == w[i]); }
}
static void test_block_two_sectors(void)
{
    pmm_init();
    virtio_blk_init();
    static uint8_t a[512], b[512], ra[512], rb[512];
    for (int i = 0; i < 512; i++) { a[i] = (uint8_t)i; b[i] = (uint8_t)(255 - i); }
    block_write(2, a); block_write(3, b);
    block_read(2, ra); block_read(3, rb);
    for (int i = 0; i < 512; i++) { KASSERT(ra[i] == a[i] && rb[i] == b[i]); }
}
```
Registry: three entries. `#include "block.h"` + `virtio_blk_init` decl.

- [ ] **Step 2 — RED** (block/virtio_blk_init undefined).

- [ ] **Step 3 — `block.h`:** `void virtio_blk_init(void); int block_present(void);
int block_read(uint64_t sector, void *buf); int block_write(uint64_t sector, const void *buf);`.

- [ ] **Step 4 — `virtio_blk.c`:**
- `virtio_blk_init`: `base = virtio_find(2)`; if 0, mark absent; else
  `virtio_setup_queue(base, &q)`; mark present; (optionally read capacity from
  config space at offset 0x100).
- block I/O helper `blk_rw(sector, buf, write)`: fill a static header
  `{ type = write?1:0, 0, sector }`, build 3 `vbuf`s (header read-only; data with
  `write = !write` i.e. device writes for a read; status `write=1`), copy `buf`
  into a static 512 bounce buffer for writes (and out of it for reads, so the DMA
  target is identity-mapped kernel RAM), `virtq_submit`, check status byte == 0.
- `block_read`/`block_write` wrap `blk_rw`. `block_present` returns the flag.

- [ ] **Step 5 — `kmain.c`:** call `virtio_blk_init()` after `pmm_init`/`kheap_init`
in the normal boot path; log capacity or "no disk".

- [ ] **Step 6 — GREEN:** `make test` runs the 3 block tests against QEMU's disk.
Debug the virtqueue if the status byte isn't 0 or `used` never advances (common:
ring address split, missing `dsb`, wrong descriptor flags).

- [ ] **Step 7 — commit + push** `feat(virtio): virtio-blk read/write sectors (Phase 19, test-first)`.
