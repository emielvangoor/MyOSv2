### Task 2: delivery (default terminate + handler trampoline) + sigreturn

**Files:** `src/signal.c`, `src/exceptions.c`, `src/syscall.*`, `src/tests.c`

- [ ] **Step 1 — failing test** (pure policy + frame save/restore):
```c
static void test_sig_default_vs_handler(void)
{
    pmm_init(); kheap_init(); vm_init(); sched_init();
    struct thread *t = thread_create(sig_noop, 0, 1);
    t->sig_pending = (1u << SIGTERM);
    KASSERT(signal_action(t, SIGTERM) == 0);          // 0 = default (terminate)
    t->sig_handler[SIGINT] = (uint64_t(*)(int))0x8000000040ULL;
    KASSERT(signal_action(t, SIGINT) == 0x8000000040ULL); // handler address
    KASSERT(signal_action(t, SIGKILL) == 0);          // SIGKILL uncatchable
}
```
(`signal_action(t,sig)` returns the handler VA, or 0 for default; SIGKILL always 0.)
Registry entry.

- [ ] **Step 2 — RED. Step 3 — implement** `signal_action` + `signals_deliver` in
`signal.c`:
```c
uint64_t signal_action(struct thread *t, int sig)
{
    if (sig == SIGKILL) { return 0; }            // can't be caught
    return (uint64_t)(uintptr_t)t->sig_handler[sig];
}

void signals_deliver(struct trapframe *tf)
{
    struct thread *t = sched_current();
    if (!t || !t->sig_pending) { return; }
    for (int sig = 1; sig < NSIG; sig++) {
        if (!(t->sig_pending & (1ull << sig))) { continue; }
        t->sig_pending &= ~(1ull << sig);
        uint64_t h = signal_action(t, sig);
        if (!h) { thread_exit(128 + sig); }      // default: terminate (no return)
        // handler: push the trap frame on the user stack, jump to the handler.
        uint64_t sp = (tf->sp_el0 - sizeof(struct trapframe)) & ~15ull;
        uint64_t *src = (uint64_t *)tf, *dst = (uint64_t *)(uintptr_t)sp;
        for (unsigned i = 0; i < sizeof(struct trapframe)/8; i++) { dst[i] = src[i]; }
        tf->sp_el0 = sp;
        tf->x[0] = sig;
        tf->elr = h;
        tf->x[30] = t->sig_tramp;                // handler returns into __sigreturn
        return;                                  // one signal per return
    }
}
```
`exceptions.c`: call `signals_deliver(tf)` at the end of `el0_sync_handler`'s SVC
case (after `do_syscall`) and in the IRQ path when the IRQ came from EL0 (SPSR.M
bits == 0). `syscall.c`:
- `SYS_KILL` → `sched_kill(x0, x1)`.
- `SYS_SIGNAL(sig, handler, tramp)` → `current->sig_handler[sig] = handler;
  current->sig_tramp = tramp; ret = 0`.
- `SYS_SIGRETURN` → restore: copy the saved frame from `tf->sp_el0` back into
  `*tf` (the trampoline left sp at the saved frame).

- [ ] **Step 4 — GREEN. Commit + push** `feat(sig): delivery + handlers + sigreturn (Phase 18, test-first)`.
