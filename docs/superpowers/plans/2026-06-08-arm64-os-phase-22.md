# MyOSv2 Phase 22 — TCP/IP stack + sockets Plan (capstone)

> REQUIRED SUB-SKILL: superpowers:executing-plans. Checkbox steps.

**Goal:** ARP → IPv4/ICMP (user-space ping) → UDP sockets → minimal TCP client.

Reference: `docs/superpowers/specs/2026-06-08-arm64-os-phase-22-design.md`.

All stack state lives in `net.c`; the kernel **net thread** polls `net_recv` and
dispatches. Constants: our IP `10.0.2.15`, gateway `10.0.2.2`. Build incrementally,
committing + pushing each task.

### Task 1: checksum + net core + ARP

**Files:** `src/net.h` (extend), `src/netstack.c`, `src/sched.c`, `src/kmain.c`, `src/tests.c`

- [ ] **Step 1 — failing tests:**
```c
static void test_inet_checksum(void)
{
    uint8_t d[20] = {0x45,0,0,0x1c, 0,0,0,0, 0x40,1,0,0, 10,0,2,15, 10,0,2,2};
    KASSERT(inet_csum(d, 20) != 0);               // some non-trivial value
    // checksum then verify-as-zero property
    uint16_t c = inet_csum(d, 20);
    d[10] = (uint8_t)(c >> 8); d[11] = (uint8_t)(c & 0xff);
    KASSERT(inet_csum(d, 20) == 0);
}
static void test_arp_resolve(void)
{
    pmm_init(); kheap_init(); vm_init(); sched_init();
    virtio_net_init(); net_stack_init();
    uint8_t mac[6];
    KASSERT(arp_resolve(0x0a000202, mac) == 0);   // 10.0.2.2 -> gateway MAC
    KASSERT(mac[0] | mac[1] | mac[2] | mac[3] | mac[4] | mac[5]);
}
```
(`net_stack_init` starts the net thread; `arp_resolve` yields while it runs.
IPs are host-order `uint32_t`; `0x0a000202` = 10.0.2.2.)

- [ ] **Step 2 — RED. Step 3 — implement `netstack.c`:**
- `inet_csum(buf, len)` — one's-complement sum, fold, invert.
- stack state + `net_stack_init()` → set MAC (from `net_mac`), start
  `thread_create(net_thread, 0, prio)`. `net_thread` loops:
  `n = net_recv(frame, max); if (n>0) net_input(frame, n); else yield();`.
- `net_input`: ethertype 0x0806 → `arp_input`; 0x0800 → `ip_input` (Task 2).
- ARP cache (8 entries); `arp_input` caches replies + answers requests for our IP;
  `arp_resolve(ip, out)`: cache hit → copy; else build+`net_send` an ARP request,
  then yield-poll the cache up to ~2s of ticks; -1 on timeout.
- Ethernet send helper `eth_send(dst_mac, ethertype, payload, len)`.

- [ ] **Step 4 — kmain:** `net_stack_init()` after `virtio_net_init`.
- [ ] **Step 5 — GREEN. Commit + push** `feat(net): checksum + net thread + ARP (Phase 22, test-first)`.

### Task 2: IPv4 + ICMP ping

**Files:** `src/netstack.c`, `src/net.h`, `src/syscall.*`, `user/*`, `src/tests.c`

- [ ] **Step 1 — failing test:**
```c
static void test_icmp_ping(void)
{
    pmm_init(); kheap_init(); vm_init(); sched_init();
    virtio_net_init(); net_stack_init();
    int ms = -1;
    KASSERT(net_ping(0x0a000202, &ms) == 0);      // gateway replies to echo
}
```

- [ ] **Step 2 — RED. Step 3 — implement:**
- `ip_send(dst_ip, proto, payload, len)`: next-hop = gateway (or dst if on the
  10.0.2.0/24 subnet); `arp_resolve` it; build the IPv4 header (version/ihl, total
  len, ttl 64, proto, `inet_csum`), then `eth_send` 0x0800.
- `ip_input`: check version/checksum; proto 1 → `icmp_input`, 17 → `udp_input`
  (Task 3), 6 → `tcp_input` (Task 4).
- `icmp_input`: echo-request → reply (swap, recompute checksum, `ip_send`);
  echo-reply → record id/seq + set a "got reply" flag the waiter polls.
- `net_ping(ip, &ms)`: send an echo request (unique id/seq), yield-poll for the
  reply (timeout). `*ms` from `jiffies` delta (TIMER_HZ=1000).
- `SYS_PING(ip, &ms)` → `net_ping`. ulib `ping()`. `/bin/ping` pings 10.0.2.2 and
  prints the time. Add to PROGS + initrd.

- [ ] **Step 4 — GREEN + live:** `make test` green; `make run` then `ping` prints a
reply. **Commit + push** `feat(net): IPv4 + ICMP + user-space /bin/ping (Phase 22)`.

### Task 3: UDP + sockets

**Files:** `src/netstack.c`, `src/net.h`, `src/syscall.*`, `src/vfs.h` (socket fd), `user/*`, `src/tests.c`

- [ ] **Step 1 — failing test** (UDP send/recv via a DNS query to 10.0.2.3:53, or
assert the socket machinery if no responder):
```c
static void test_udp_roundtrip(void)
{
    pmm_init(); kheap_init(); vm_init(); sched_init();
    virtio_net_init(); net_stack_init();
    int s = sock_udp();
    sock_bind(s, 12345);
    uint8_t q[] = { /* minimal DNS query for example.com A */ ... };
    KASSERT(sock_sendto(s, q, sizeof(q), 0x0a000203, 53) == (int)sizeof(q));
    uint8_t r[512]; uint32_t fip; uint16_t fport;
    int n = sock_recvfrom(s, r, sizeof(r), &fip, &fport);   // yields for the reply
    KASSERT(n > 0);
}
```
(Kernel-level socket API `sock_*`; the syscall layer wraps it.)

- [ ] **Step 2 — RED. Step 3 — implement:** a socket table (UDP: local port + a
small datagram queue). `udp_input` enqueues to the matching port. `sock_sendto`
builds UDP+IP+Eth. `sock_recvfrom` yield-polls the queue. Syscalls
`SYS_SOCKET/BIND/SENDTO/RECVFROM`; a socket is held in the fd table (a new
`struct file` kind, like a pipe). ulib wrappers.

- [ ] **Step 4 — GREEN. Commit + push** `feat(net): UDP + sockets API (Phase 22, test-first)`.

### Task 4: minimal TCP client

**Files:** `src/netstack.c`, `src/net.h`, `src/syscall.*`, `src/tests.c`

- [ ] **Step 1:** a single-connection TCP state machine: `tcp_connect(ip, port)`
(SYN → SYN/ACK → ACK), `tcp_send`/`tcp_recv` (seq/ack + retransmit timeout),
`tcp_close` (FIN). `tcp_input` drives the state machine. `SYS_CONNECT`; `send`/
`recv`/`close` over the socket fd via `SYS_WRITE`/`READ`/`CLOSE` routing.
- [ ] **Step 2 — test:** connect to QEMU's gateway forwarded port, or assert the
handshake state transitions on injected segments (unit-level) if no live server.
- [ ] **Step 3 — commit + push** `feat(net): minimal TCP client (Phase 22, test-first)`.
  (If a live TCP peer isn't reachable from the test harness, keep TCP unit-tested
  at the state-machine level and document the limitation.)
