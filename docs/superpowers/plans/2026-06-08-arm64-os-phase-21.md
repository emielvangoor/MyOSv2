- [ ] **Step 2 — transport refactor (`virtio.c`/`.h`):** split `virtio_setup_queue`
into:
- `int virtio_init(uint64_t base)` — reset, ACK|DRIVER, negotiate VERSION_1, FEATURES_OK.
- `int virtio_queue_init(uint64_t base, struct virtq *q, int index)` — QueueSel=index,

  `virtio_driver_ok` (keeps virtio-blk working).

`R_QUEUE_NOTIFY = q->notify_idx`.

### Task 2: virtio-net driver

**Files:** `src/net.h`, `src/virtio_net.c`, `src/kmain.c`

- [ ] **Step 1 — `net.h`:** `void virtio_net_init(void); int net_present(void);
void net_mac(uint8_t out[6]); int net_send(const void *frame, int len);
int net_recv(void *buf, int max);`.

- [ ] **Step 2 — `virtio_net.c`:**
- `struct virtio_net_hdr { uint8_t flags, gso_type; uint16_t hdr_len, gso_size,
  csum_start, csum_offset, num_buffers; };` (12 bytes).
- statics: `nic_base`, `struct virtq rxq, txq`, `nic_ok`, `uint8_t mac[6]`.
- RX buffers: `static uint8_t rxbuf[NRX][2048];` (header + frame). On init, post
  each as a device-writable descriptor on rxq (one buf per descriptor), and track
  which used entries we've consumed.
- `virtio_net_init`: `nic_base = virtio_find(1)`; `virtio_init`;
  `virtio_queue_init(rxq,0)`; `virtio_queue_init(txq,1)`; `virtio_driver_ok`; read
  `mac` from `nic_base + 0x100`; post all RX buffers; `nic_ok = 1`.
- `net_send`: a static `txbuf[2048]`: zero a `virtio_net_hdr` at the front, copy
  the frame after it, submit `{txbuf, sizeof(hdr)+len, write=0}` on txq, poll.
- `net_recv`: check `rxq.used[1] != rxq.last_used`; if so, read the used entry
  `{id, len}`, copy `rxbuf[id] + sizeof(hdr)` (len - hdr) into `buf`, **re-post**
  that buffer onto rxq (available ring), advance `last_used`, return the payload
  length; else 0.
- `net_present`/`net_mac` accessors.

  (RX posting needs `virtq` to expose adding a single device-writable descriptor
  to the available ring without the submit-and-poll of `virtq_submit`. Add a small
  `virtq_add_recv(q, addr, len)` helper in `virtio.c`, or post directly here using
  the queue fields. Keep one buffer per descriptor index = ring slot.)

- [ ] **Step 3 — `kmain.c`:** `virtio_net_init()`; log
  `net: <mac> ready` / `net: none`.

- [ ] **Step 4 — build + boot:** MAC logged (e.g. `52:54:00:12:34:56`).
Commit + push `feat(net): virtio-net RX/TX + MAC (Phase 21)`.

### Task 3: ARP round-trip test

**Files:** `src/tests.c`

- [ ] **Step 1 — tests:**
```c
static void test_net_present(void)
{
    pmm_init(); kheap_init(); virtio_net_init();
    KASSERT(net_present());
    uint8_t m[6]; net_mac(m);
    KASSERT(m[0] | m[1] | m[2] | m[3] | m[4] | m[5]);   // non-zero MAC
}
static void test_net_arp_roundtrip(void)
{
    pmm_init(); kheap_init(); virtio_net_init();
    uint8_t mac[6]; net_mac(mac);
    // Build an ARP request: who-has 10.0.2.2, tell 10.0.2.15.
    uint8_t f[42];
    for (int i=0;i<6;i++) f[i]=0xff;              // dst: broadcast
    for (int i=0;i<6;i++) f[6+i]=mac[i];          // src: us
    f[12]=0x08; f[13]=0x06;                       // ethertype ARP
    uint8_t arp[28] = {0,1, 8,0, 6,4, 0,1,        // hw eth, proto ipv4, opcode request
        mac[0],mac[1],mac[2],mac[3],mac[4],mac[5],10,0,2,15,   // sender
        0,0,0,0,0,0, 10,0,2,2};                   // target
    for (int i=0;i<28;i++) f[14+i]=arp[i];
    KASSERT(net_send(f, 42) == 0);
    // Poll for the reply (opcode 2 from 10.0.2.2).
    uint8_t r[1600]; int got = 0;
    for (int tries=0; tries<2000000 && !got; tries++) {
        int n = net_recv(r, sizeof(r));
        if (n >= 42 && r[12]==0x08 && r[13]==0x06 && r[14+7]==2 &&
            r[14+24]==10 && r[14+25]==0 && r[14+26]==2 && r[14+27]==2) { got = 1; }
    }
    KASSERT(got);
}
```
Registry: two entries. `#include "net.h"`.
